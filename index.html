Available endpoints
GET /api/health GET

Returns api status.

GET /api/suppliers/quality GET

Aggregated reject rate per supplier based on Serial and Inspection.

GET /api/serial/:serial/trace GET

Trace a serial back to its batch and supplier, plus other serials from that batch.

GET /api/serial/random GET

Return ten random serial numbers to make testing easier.

POST /api/admin/clear POST

Clear all data in the Neo4j database.

POST /api/admin/seed POST

Create the random demo dataset.
